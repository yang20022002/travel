开发工具：
开发语言为 Java、HTML、CSS、JavaScript，用于实现前后端功能开发。
集成开发环境使用 IDEA，负责项目代码编写、编译与调试。
项目管理工具采用 Maven，实现依赖包管理与项目构建。
Web 服务器选用 Tomcat，用于部署并运行 Web 项目。
数据库使用 MySQL 和 Redis，MySQL 存储业务数据，Redis 提升数据访问效率。
接口测试工具使用 Postman，测试后端 API 接口可用性。

细节：
1.前后端交互技术
采用 AJAX 异步请求方式实现用户注册功能，前端以 JSON 格式提交数据至 /signup 接口，后端处理后返回结果，提升用户交互体验。
2.数据校验机制
前端校验：通过 JavaScript 函数验证用户名（5-20 位）、密码（6-20 位）、邮箱格式等字段有效性。
后端校验：在 RegisterController.java 中校验数据非空性，判断用户名与邮箱是否已存在，确保数据合规性。
3.用户状态管理
注册用户默认设为未激活状态（status='N'），生成 20 位随机激活码，控制台输出激活链接，为后续用户激活功能提供支持。
4.项目构建与发布
配置 clean-build-war 和 redeploy 任务，实现项目的快速构建与重新发布，简化开发调试流程。
